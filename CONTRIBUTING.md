## Style guides

### Git Commit Messages

* Use the present tense ("Add feature" not "Added feature")
* Use the imperative mood ("Move cursor to..." not "Moves cursor to...")

### Java Code Style Guide

* Use [Google Java Format](https://github.com/google/google-java-format)
